## Quick Start

Dependencies:
- [GLFW](https://www.glfw.org/)
- [stb](https://github.com/nothings/stb)
- [cglm](https://www.glfw.org/)
- [glad](https://glad.dav1d.de/)



```console
$ make install
$ make file=Camera.c output=Camera all
```
